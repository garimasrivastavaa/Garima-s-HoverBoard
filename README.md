# Garima's HoverBoard

## Project Overview
Garima's HoverBoard is an interactive web project crafted with HTML, CSS, and JavaScript. It features a dynamic, user-responsive hoverboard interface with real-time visual effects, showcasing advanced web development techniques and creativity in design.

![image](https://github.com/user-attachments/assets/9f267e59-6d74-42e3-be7b-1cfd5702e9f5)

## Features
- **Dynamic Interaction**: Experience the hover effects as you interact with the board.
- **Real-Time Visual Effects**: Watch as the hoverboard responds with vibrant visual feedback.
- **Responsive Design**: Designed to be fully responsive, providing an engaging experience on various devices and screen sizes.

## Technologies Used
- **HTML**: Structures the web content and layout.
- **CSS**: Styles the presentation, including colors and fonts.
- **JavaScript**: Powers the interactivity and dynamic effects of the hoverboard.

## Setup and Installation
To get this project up and running on your local machine, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/garimasrivastavaa/Garima-s-HoverBoard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Garima-s-HoverBoard
   ```
3. Open the `index.html` in your preferred web browser.

## How to Use
Simply move your cursor over the hoverboard to activate the effects. The squares will change colors, demonstrating a range of visual feedback based on interaction.
